# TUSG コーポレートサイト

## プロジェクト概要

**TUSG（合同会社タス）**のコーポレートウェブサイトです。営業代行・マーケティング支援サービスを提供する企業の公式サイトとして、成果を重視したビジネスモデルと信頼性を訴求するデザインを実現しています。

## 🎯 サイト目的

- 売上アウトソーシングプロジェクトの獲得（企業・個人事業主向け）
- 金融機関への信用補完とTUSGグループ基盤構築
- コンタクトフォーム経由の問い合わせ獲得
- 営業アポイントメントの取得

## 👥 ターゲットユーザー

### 主要ターゲット
- **売上アウトソーシング代理店**（店舗型ビジネス向け）
- **マーケティング・広告代理店**
- **店舗向けプロダクト企業**

### 想定来訪者
- **発注担当者**（30-50代）
- **企業の審査担当者**
- **将来のパートナー候補**

## 📄 実装済みページ一覧

### 1. トップページ (index.html)
- **ファーストビュー**: 「成果を足す。未来を足す。」のメインビジュアル
- **セクション構成**:
  - TUSGとは（企業理念）
  - 事業内容（3つのサービス）
  - 強み・選ばれる理由（3つの強み）
  - 想定実績モデル（月商200万円モデル）
  - お問い合わせCTA

### 2. 事業内容ページ (business.html)
- **成果報酬型営業代行事業**（主力事業）
- **マーケティング支援事業**
- **営業戦略設計事業**

### 3. 強み・選ばれる理由ページ (strengths.html)
- **実績に裏付けされた営業力**
- **営業×マーケティングの融合**
- **成果に責任を持つスタンス**

### 4. 代表挨拶・会社理念ページ (about.html)
- 「関わるすべての人の人生にプラスを」という理念
- 代表社員からのメッセージ

### 5. 会社概要ページ (company.html)
- 会社基本情報（設立、所在地、事業内容など）

### 6. お問い合わせページ (contact.html)
- コンタクトフォーム（会社名、氏名、メール、電話、問い合わせ種別、内容）
- 24時間以内返信対応の明記
- 個人情報保護の説明

## 🎨 デザイン仕様

### カラースキーム
- **メインカラー**: グリーン（#2E8B57）
- **セカンダリ**: 白・グレー
- **アクセント**: ブラック
- **背景**: ライトグレー（#F5F5F5）

### デザインテイスト
- シンプルで信頼感のあるデザイン
- ベンチャーらしさを残しつつ、軽すぎない落ち着いた印象
- レスポンシブ対応（モバイル・タブレット・PC）
